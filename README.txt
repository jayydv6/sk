==========================================
        GUESSER SYSTEM
==========================================

HOW IT WORKS:
1. Admin creates guesser accounts from admin panel
2. Guessers login at guesser.html with ID:password
3. Guessers can submit max 3 numbers (00-99) per game
4. All predictions appear on main page
5. Admin can manage guesser status

GUESSER RULES:
- Only numbers 00-99 allowed
- Maximum 3 numbers per prediction
- One prediction per game per day
- Predictions visible to all users

==========================================
        TROUBLESHOOTING
==========================================

PROBLEM: Data not saving
SOLUTION: Clear browser cache and reload

PROBLEM: Admin login not working
SOLUTION: Check credentials in admin-script.js

PROBLEM: Ads not displaying
SOLUTION: Check ad code format in admin panel

PROBLEM: Mobile display issues
SOLUTION: Ensure viewport meta tag present

PROBLEM: Numbers not accepting 00-99
SOLUTION: Use number inputs with min="0" max="99"
